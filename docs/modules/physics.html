Overview

The ar-physics module provides comprehensive abstractions for quantum and classical physics simulation. It includes atomic structure modeling with electrons, orbitals, and shells, photon field simulation for light propagation, and rigid body dynamics for classical mechanics.

Atomic Structure

Model atoms with electron shells, orbitals, and quantum mechanics. Supports electron configuration, spin states, and energy level transitions.

Quantum Mechanics

Electron spin states (Up/Down), orbital energy levels, excitation energies, and quantum number-based orbital definitions (s, p, d, f).

Photon Fields

Simulate light propagation in 3D space with photon absorption and emission. Time-stepped simulation with Clock interface.

Rigid Body Dynamics

Classical mechanics state management with position, velocity, force application, and time-based integration.

Module Architecture

Atomic Structure Components

Atom

Complete atomic model with atomic number and electron shells. Foundation for molecular and chemical simulations.

Shell

Electron shell container (K, L, M, N). Manages orbital occupancy and electron capacity per shell.

Orbital

Quantum orbital with energy levels. Supports s, p, d, f orbital types with proper quantum numbers.

Electron

Individual electron with spin state and excitation energy. Tracks quantum state transitions.

Quantum Mechanics

Spin

Electron spin enumeration (Up/Down). Fundamental quantum property for Pauli exclusion principle.

PhysicalConstants

Standard physics constants: Planck's constant (H), speed of light (C), Rydberg constant (R), and unit conversions.

Photon Simulation

PhotonField

Manages photons in 3D space. Supports adding photons with position, direction, and energy in eV.

Absorber

Interface for objects that absorb and emit photons. Used for cameras, surfaces, and detectors.

Clock

Time-stepped simulation controller. Manages photon fields and advances simulation time.

Rigid Body Dynamics

RigidBody.State

Physics state container with position, velocity, mass, and force accumulation. Supports time integration.

Core Concepts Deep Dive

Atom and Electron Configuration

The Atom class represents an atomic model with electron shells following quantum mechanical principles.

// Create nitrogen atom (atomic number 7)
Atom nitrogen = new Atom(7, Arrays.asList(
    Shell.first(2),        // K shell: 1s^2
    Shell.second(2, 3)     // L shell: 2s^2 2p^3
));

// Access valence electrons
Shell valenceShell = nitrogen.getValenceShell();
Electrons electrons = valenceShell.getElectrons();

Orbital Energy Levels

Orbitals define quantum states with specific energy levels based on principal and angular momentum quantum numbers.

// Access predefined orbitals
Orbital orbital_1s = Orbital.s1;    // 1s orbital
Orbital orbital_2s = Orbital.s2;    // 2s orbital
Orbital orbital_2px = Orbital.p2x;  // 2p orbital (x direction)

// Get orbital energy
double energy = orbital_1s.getEnergy();

Electron Spin and Excitation

Electrons have spin states and can absorb energy to reach excited states.

// Create electron with spin up
Electron e = new Electron(Spin.Up);

// Add excitation energy (in eV)
double excitationEv = 10.2;  // Hydrogen Lyman-alpha energy
e.addExcitationEnergy(excitationEv);

// Check spin state
Spin currentSpin = e.getSpin();

PhotonField Interface

Manages photon propagation and interaction in 3D space.

public interface PhotonField {
    // Add a photon to the field
    void addPhoton(Vector position, Vector direction, double energyEv);

    // Advance simulation by one time step
    void tick();

    // Get all photons in the field
    Collection<Photon> getPhotons();
}

Absorber Interface

Objects that can absorb and emit photons implement this interface.

public interface Absorber {
    // Absorb an incoming photon
    void absorb(Photon photon);

    // Emit stored energy as photons
    void emit();

    // Get accumulated emission energy
    double getEmitEnergy();
}

Rigid Body State

Classical mechanics state management for physics simulations.

// Create rigid body state
RigidBody.State body = new RigidBody.State();

// Set physical properties
body.setMass(1.0);                    // 1 kg
body.setPosition(new Vector(0, 0, 0));
body.setVelocity(new Vector(0, 0, 0));

// Apply forces
body.applyForce(new Vector(0, -9.8, 0));  // Gravity
body.applyTorque(torqueVector);

// Integrate over time
body.update(deltaTime);

// Get updated state
Vector newPosition = body.getPosition();
Vector newVelocity = body.getVelocity();

Usage Examples

Atomic Spectroscopy Simulation

public class SpectroscopySimulator {
    public void simulateHydrogenAbsorption() {
        // Create hydrogen atom (1 electron in 1s orbital)
        Atom hydrogen = new Atom(1, List.of(Shell.first(1)));

        // Get valence electrons
        Electrons electrons = hydrogen.getValenceShell().getElectrons();

        // Attempt photon absorption
        double photonEnergyEv = 10.2;  // Lyman-alpha
        boolean absorbed = electrons.absorb(photonEnergyEv);

        if (absorbed) {
            System.out.println("Photon absorbed - electron excited");
        }
    }
}

Photon Field Simulation

public class LightSimulation {
    public void simulatePhotonPropagation() {
        // Create photon field
        PhotonField field = new PhotonFieldImpl();

        // Create absorber (e.g., camera sensor)
        Absorber camera = new PinholeCameraAbsorber();

        // Add photons to the field
        Vector position = new Vector(0, 0, 0);
        Vector direction = new Vector(1, 0, 0);
        double energyEv = 2.0;  // Visible light (~620 nm)
        field.addPhoton(position, direction, energyEv);

        // Create simulation clock
        Clock clock = new Clock();
        clock.addPhotonField(field);

        // Run simulation
        int steps = 1000;
        for (int i = 0; i < steps; i++) {
            clock.tick();
            camera.absorb(field);
        }

        // Process absorbed photons
        double capturedEnergy = camera.getEmitEnergy();
    }
}

Rigid Body Physics

public class PhysicsSimulator {
    private static final double GRAVITY = 9.8;

    public void simulateProjectile() {
        // Create projectile
        RigidBody.State projectile = new RigidBody.State();
        projectile.setMass(0.5);  // 500g
        projectile.setPosition(new Vector(0, 10, 0));  // Start at 10m height
        projectile.setVelocity(new Vector(5, 0, 0));   // 5 m/s horizontal

        // Simulation parameters
        double deltaTime = 0.01;  // 10ms timestep
        double time = 0;

        // Run until projectile hits ground
        while (projectile.getPosition().getY() >= 0) {
            // Apply gravity
            Vector gravityForce = new Vector(0, -GRAVITY * projectile.getMass(), 0);
            projectile.applyForce(gravityForce);

            // Update physics
            projectile.update(deltaTime);
            time += deltaTime;

            System.out.printf("t=%.2f: pos=(%.2f, %.2f, %.2f)%n",
                time,
                projectile.getPosition().getX(),
                projectile.getPosition().getY(),
                projectile.getPosition().getZ());
        }
    }
}

Building Complex Atoms

public class AtomBuilder {
    public Atom createCarbon() {
        // Carbon: 1s^2 2s^2 2p^2
        return new Atom(6, Arrays.asList(
            Shell.first(2),        // K shell: 2 electrons
            Shell.second(2, 2)     // L shell: 2 in s, 2 in p
        ));
    }

    public Atom createOxygen() {
        // Oxygen: 1s^2 2s^2 2p^4
        return new Atom(8, Arrays.asList(
            Shell.first(2),        // K shell: 2 electrons
            Shell.second(2, 4)     // L shell: 2 in s, 4 in p
        ));
    }

    public Atom createIron() {
        // Iron: 1s^2 2s^2 2p^6 3s^2 3p^6 3d^6 4s^2
        return new Atom(26, Arrays.asList(
            Shell.first(2),           // K: 2
            Shell.second(2, 6),       // L: 8
            Shell.third(2, 6, 6),     // M: 14
            Shell.fourth(2, 0, 0, 0)  // N: 2 (4s^2)
        ));
    }
}

Using Physical Constants

public class PhysicsCalculations {
    public double wavelengthToEnergy(double wavelengthNm) {
        // E = hc / wavelength
        double wavelengthM = wavelengthNm * 1e-9;
        double energyJoules = (PhysicalConstants.H * PhysicalConstants.C) / wavelengthM;
        return energyJoules;
    }

    public double energyToWavelength(double energyEv) {
        // wavelength = hc / E
        double energyJoules = PhysicalConstants.eVtoWatts(energyEv);
        double wavelengthM = (PhysicalConstants.H * PhysicalConstants.C) / energyJoules;
        return wavelengthM * 1e9;  // Convert to nm
    }

    public double hydrogenTransitionEnergy(int nLower, int nUpper) {
        // Rydberg formula: E = R * (1/n1^2 - 1/n2^2)
        return PhysicalConstants.R * (1.0/(nLower*nLower) - 1.0/(nUpper*nUpper));
    }
}
