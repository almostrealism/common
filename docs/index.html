Overview

Almost Realism is a comprehensive Java framework for building hardware-accelerated applications in machine learning, computer graphics, and physics simulation. The framework provides:

Hardware Acceleration

Automatic GPU compilation with support for OpenCL, Metal, and native JNI backends.

ML & Deep Learning

Transformer models, autoregressive inference, and training infrastructure.

Physics Simulation

Quantum mechanics, photon fields, and rigid body dynamics.

Computational Geometry

Vector math, ray tracing, and 3D graphics primitives.

Getting Started

Installation

<dependency>
    <groupId>org.almostrealism</groupId>
    <artifactId>ar-{module}</artifactId>
    <!-- Check pom.xml for current version -->
</dependency>

Environment Setup

For hardware acceleration, set environment variables:

export AR_HARDWARE_LIBS=/tmp/ar_libs/
# AR_HARDWARE_DRIVER is best left unset to auto-detect the best available backend

Quick Example

import org.almostrealism.algebra.*;
import org.almostrealism.collect.*;

// Create vectors
PackedCollection vectors = Vector.bank(1000);

// Hardware-accelerated operations
CollectionProducer<?> normalized = normalize(cp(vectors));
CollectionProducer<?> scaled = normalized.multiply(2.0);

// Execute on GPU
PackedCollection result = scaled.get().evaluate();
