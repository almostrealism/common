Buffer consolidation, warmup strategy, and test coverage for real-time AudioScene

Implements the remaining goals from the REALTIME_AUDIO_SCENE plan:

1. PatternAudioBuffer output buffer consolidation: Pre-allocates a single
   contiguous PackedCollection for all render cell output buffers. Each
   PatternAudioBuffer receives a delegate range into the consolidated
   buffer. Scope argument deduplication collapses all render buffers
   into a single kernel argument (24 buffers -> 1 for 6 channels).

2. EfxManager filter buffer consolidation: Pre-allocates a single buffer
   for all filter destination buffers in applyFilter(). Each wet channel's
   filter destination is a delegate range (up to 12 buffers -> 1).

3. Warmup strategy: AudioScene.warmNoteCache() pre-evaluates all note
   audio producers to populate the FrequencyCache before the real-time
   loop starts, eliminating first-encounter compilation spikes.

4. New test class AudioSceneBufferConsolidationTest with 5 tests:
   - renderBufferConsolidation: verifies consolidated buffer structure
   - consolidatedBuffersProduceValidAudio: correctness after consolidation
   - genomeIndependence: genome change reflected without recompilation
   - effectsEnabledPerformance: timing with full effects pipeline (6ch)
   - warmNoteCacheReducesCompilationCost: warmup pre-evaluates notes
